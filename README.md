## Requirement

- Laser detector module
- Laser diode
- LCD Display 2004 with LCD to IIC module
- Microcontroller
### Software
- Arduino IDE
- PlatformIO *(Recommend)*
### Library
- LiquidCrystal_I2C by [Frank de Brabander](https://github.com/fdebrabander/Arduino-LiquidCrystal-I2C-library)
